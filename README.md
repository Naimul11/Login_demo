# How to run:

1. open folder in cmd.
2. python app.py
3. can see some link 
     * Running on all addresses (0.0.0.0)
     * Running on http://127.0.0.1:5000
     * Running on http://192.168.0.103:5000

4. open file in vs code.
5. run the main.dart in an emulator.
6. A login page will appear.
7. Enter credentials
